Ledger Overview

  • Title: "GCITChain Ledger"
  • Description: "The GCITChain Ledger provides a transparent, immutable record of all transactions and data entries made within the GCIT blockchain. Each entry is securely timestamped and linked to the previous one, ensuring the integrity and security of your academic data."

Transaction History

  • Search/Filter Options:
  • Search Bar: "Search by Transaction ID, User, or Date"
  • Filters: Dropdowns to filter by date range, type of transaction, or status.
  • Example:
  • Date Range: "From [Start Date] to [End Date]"
  • Transaction Type: "Data Upload, Access Request, Data Deletion"
  • Status: "Confirmed, Pending, Rejected"

Ledger Summary

  • Statistics:
  • Total Transactions: Display the total number of transactions recorded.
  • Confirmed Transactions: Number of confirmed transactions.
  • Pending Transactions: Number of transactions still pending confirmation.
